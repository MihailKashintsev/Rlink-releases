# Rlink Releases

Скачай последнюю версию Rlink для своей платформы.

| Платформа | Файл |
|-----------|------|
| Android | `.apk` |
| macOS | `_macos.zip` |
| Windows | `_windows.zip` |
| Linux | `_linux.tar.gz` |

## Установка

**Android:** скачай APK → разреши установку из неизвестных источников → установи.
**macOS:** распакуй ZIP → перетащи в Applications.
**Windows:** распакуй ZIP → запусти Rlink.exe.
**Linux:** распакуй tar.gz → запусти rlink.
